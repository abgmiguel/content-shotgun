# Layer 2: Workspace Contract

## Scope
Workspace-level collaboration context and execution boundaries.

## Required Inputs
- `<workspace-parent>/instructions.md`
- `<workspace-parent>/workspaces/<workspace-slug>/knowledge/workspace.md`
- Relevant supporting files in `knowledge/`

## Requirements
1. Resolve active workspace before writing.
2. Use workspace brief as brand/product/compliance source of truth.
3. Write only inside the active topic folder under the resolved workspace.
